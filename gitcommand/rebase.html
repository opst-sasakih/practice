How to rebase

  • ①Checkout&Pull develop branch
    • git checkout develop
    • git pull origin develop
  • ②Rebase feature branch
    • git checkout feature
    • git rebase develop
  • ※When a conflict occurs
    • merge file
    • git add file
    • git rebase --continue
  • ③Push feature branch
    • git push -f origin feature
